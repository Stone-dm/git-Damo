## 五、⚠️ 注意事项

1. **嵌入模型**：`BAAI/bge-large-zh-v1.5` 首次启动需要从镜像下载（约 1.3GB），已在 `app/core/embedding.py` 中配置国内镜像 `hf-mirror.com`
2. **DeepSeek API**：需要有效的 API Key，已配置在 `.env` 文件中
3. **MySQL**：每个组员需有自己的 MySQL 服务，创建 `party_building` 数据库
4. **Redis**：已安装的可直接用，未安装的可在 Docker 中运行
5. **Docker**：最终提交前需完整测试 `docker-compose up -d` 流程

## 七、🚀 快速启动命令

```bash
# 本地开发
conda activate party-building
uvicorn app.main:app --host 0.0.0.0 --port 8000
# 访问 http://localhost:8000/docs 或 http://localhost:8000/ui/

# Docker 部署
docker-compose up -d
docker-compose exec app python scripts/init_db.py
docker-compose exec app python scripts/seed_data.py
docker-compose exec app python scripts/ingest_docs.py
```
